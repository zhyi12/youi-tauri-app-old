## What ?

    基于tauri和svelte ，使用Rust和TypeScript语言开发的桌面应用项目

## 使用的技术

### 框架
* [tauri](https://tauri.app/)
* [svelte](https://svelte.dev/)
* [svelte-kit](https://kit.svelte.dev/)
* [sveltestrap](https://sveltestrap.js.org/)
* [carbon-components-svelte](https://carbon-components-svelte.onrender.com/)

### tauri插件
* [tauri-plugin-sql](https://github.com/tauri-apps/tauri-plugin-sql)
* [tauri-plugin-fs-watch](https://github.com/tauri-apps/tauri-plugin-fs-watch)
* [tauri-plugin-fs-extra](https://github.com/tauri-apps/tauri-plugin-fs-extra)
* [tauri-plugin-fs-log](https://github.com/tauri-apps/tauri-plugin-log)
